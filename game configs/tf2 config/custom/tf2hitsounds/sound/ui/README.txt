To install go into Tf2s files usually by following these files

(Pc you use) -> SteamLibrary -> Steamapps -> common -> Team Fortress 2 -> tf 

Next youll need to create a couple files in the tf folder

1. First make a new folder called "tf2hitsounds" (without "") *NOTE* it has to be named exactly the way its typed**

2. Inside the tf2hitsounds folder make a new one called "sound"

3. Inside the sound folder make a new one called "ui"

4. Inside the ui folder just drag and drop the Hitsound and Killsound folder and your good to go!
